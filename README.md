# restaurant-landing-page
# 🍽️ Restaurant Landing Page

A clean, single-page HTML/CSS restaurant website skeleton structured with all the essential sections needed for a modern restaurant web presence.

---

## 📁 File Structure

```
restaurant-landing/
│
├── index.html   # Main HTML file (all sections)
│
├── styles.css                 #project styling
└── README.md                 # Project documentation
```

---

## 📄 Page Sections

The page is divided into the following sections, in order:

### 1. Navigation Bar
- Fixed to the top of the page so it stays visible while scrolling
- Contains a logo placeholder on the left
- Anchor links on the right that jump to each section of the page
- Dark background (`#222`) for contrast against page content

### 2. Hero
- Full-viewport-height section
- Intended for a headline, tagline, background image, and a call-to-action button (e.g. "Reserve a Table")

### 3. About
- Tells the restaurant's story — its history, values, and what makes it unique
- Can include an image alongside a block of text

### 4. Menu
- Showcases the food and drink offerings
- Can be organized into categories (Starters, Mains, Desserts, Drinks)

### 5. Chef's Specials
- Highlights featured or seasonal dishes
- Great for promotions and drawing attention to signature items

### 6. Gallery
- A visual showcase of food, ambiance, and the dining experience
- Typically implemented as a grid or masonry image layout

### 7. Testimonials
- Displays customer reviews and ratings
- Builds trust and social proof for new visitors

### 8. Reservations
- A booking form section where customers can request a table
- Typically includes fields for name, date, time, party size, and contact info

### 9. Contact
- Restaurant address, phone number, email, and opening hours
- Can include an embedded Google Map

### 10. Footer
- Bottom bar with copyright info, social media links, and quick nav links

---

## 🎨 Styling Overview

- All styles are written inline within a `<style>` block in the `<head>` — no external stylesheet needed
- Sections alternate between two soft background tones (`#fff` and `#faf7f2`) to create visual separation
- The navbar and footer share a dark theme (`#222`) for a clean, framing effect
- Each section has a minimum height of `100vh` so it fills the screen

## 🚀 Getting Started

1. Download or clone the project
2. Open `restaurant-landing-page.html` in any web browser — no server or build tools required
3. Replace the placeholder text in each section with your actual content
4. Add images, fonts, and additional CSS as needed to match your brand
